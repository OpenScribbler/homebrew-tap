# OpenScribbler Homebrew Tap

Homebrew formulae for [OpenScribbler](https://github.com/OpenScribbler) projects.

## Install

```bash
brew install openscribbler/tap/phyllotaxis
```

Or tap first, then install:

```bash
brew tap openscribbler/tap
brew install phyllotaxis
```

## Update

```bash
brew upgrade phyllotaxis
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [phyllotaxis](https://github.com/OpenScribbler/phyllotaxis) | Progressive disclosure CLI for OpenAPI specs |
